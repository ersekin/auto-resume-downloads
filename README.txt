Auto Resume Downloads Local

Установка:
1. Распакуй архив в отдельную папку.
2. Открой Chrome: chrome://extensions/
3. Включи "Режим разработчика".
4. Нажми "Загрузить распакованное расширение".
5. Выбери распакованную папку auto-resume-downloads-local.
6. Запусти скачивание в браузере.

Расширение использует только разрешения downloads и alarms.
Оно не отправляет данные в интернет и не содержит внешнего кода.
